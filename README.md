# Business Productivity Hub (OrgPulse)

A centralized organizational management dashboard designed to track tasks, personnel responsibilities, and department health in real-time.

## 🚀 Key Modules
1. **Product Research**: Monitor market trends and insights.
2. **Product Development**: Track features, bugs, and builds.
3. **Product Marketing**: Campaigns and visibility tracking.
4. **Product Selling**: Sales targets and pipeline management.
5. **Accounts & Finance**: Budgeting and financial oversight.
6. **Client Success**: Retention and customer satisfaction.
7. **HR Department**: Personnel management and responsibility assignments.

## 🛠️ Tech Stack
- **Backend**: Node.js, Express
- **Database**: MongoDB (via Mongoose)
- **Frontend**: Vanilla CSS, Modern JavaScript, Lucide Icons
- **Design**: Inter & Outfit fonts, Mobile-responsive glassmorphism UI.

## 📦 Getting Started
1. **Configure MongoDB**: Update the `.env` file with your `MONGODB_URI`.
2. **Install dependencies**: `npm install`
3. **Run the server**: `npm run dev`
4. **Access**: Navigate to `http://localhost:5000` in your browser.

## ✨ Features
- **Dashboard View**: High-level overview of all departments with performance scores.
- **Summary Health View**: Quickly identify department status (Optimal, Warning, Critical).
- **HR & Personnel**: Add team members, specify roles, and assign them to tasks.
- **Audit Logs**: Full activity history of all task creates, updates, and deletes.
- **Animated UI**: Smooth transitions and modern aesthetic for a premium experience.
